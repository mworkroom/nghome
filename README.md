# NANGOK Home

`www.nangok.app`에서 4개의 업무 앱으로 이동하는 정적 메인 페이지입니다.

## 파일

- `index.html`: 페이지 구조와 앱 링크
- `style.css`: 반응형 디자인
- `favicon.svg`: 브라우저 아이콘

## 연결 주소

- 페이나우비즈: `https://paynowbiz.nangok.app`
- 가격표: `https://price.nangok.app`
- 회원 관리: `https://member.nangok.app`
- 직급 플랜: `https://plan.nangok.app`

별도의 빌드 과정이나 JavaScript 없이 바로 배포할 수 있습니다.
